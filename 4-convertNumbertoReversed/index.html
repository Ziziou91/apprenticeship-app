Question 4 - Convert number to reversed array of digits

Given a random 'number', return the digits of this number within an array in reverse order.

For example:

Given 348597 => digitise(348597) will return [7,9,5,8,4,3] .

Given 12345 => digitise(12345) will return [5,4,3,2,1] .

You can assume that n will always be a positive number.
