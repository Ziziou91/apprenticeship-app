Question 5 - Count the Characters

Complete the function countChars which takes a string and a character and returns the number of times this character is found within the string.

Think about how you will access each character of the input string in order to test whether it is the character you are looking for.

The string will only contain lowercase letters and the character you are being asked to count will always be lowercase.

For example:

Given str = 'hello' and char = 'l' => countChars('hello', 'l') will return 2 .

Given str = 'hello world' and char = 'l' => countChars('hello world', 'l') will return 3 .

Given str = 'hello world' and char = 'z' => countChars('hello world', 'z') will return 0 .
