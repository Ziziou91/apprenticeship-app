Installation Instructions

Before we can get started, there are a couple of things we need. These are:

A Text Editor - Sublime Editor
A Modern Browser - Google Chrome

Before we can being the assessment, we will work through the example first

Example - Types

Open the folder you downloaded in Sublime and work through the problems. Each question has a corresponding folder in the directory.

You can edit the 'main.js' file in each folder to write your code, and view the output of your code in 'index.html'.

The format of each question will be familiar to what you have done previously in Codewars.

You will have 1hr 15mins to work through the following questions. Online resources such as MDN and StackOverflow are allowed and encouraged.

Question 1 - Square a Number
Question 2 - Sum of Odd Numbers
Question 3 - Find the Smallest Integer
Question 4 - Convert number to reversed array of digits
Question 5 - Count the Characters
