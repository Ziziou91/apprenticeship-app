Question 2 - Sum of Odd Numbers

In this challenge, you should complete the function sumOfOdds which takes a number n and adds up all the odd numbers up to and including n.

For example:

Given 5, sumOfOdds(5) will return 9 .

Given 12, sumOfOdds(12) will return 36 .

You can assume that n will always be a positive number.
